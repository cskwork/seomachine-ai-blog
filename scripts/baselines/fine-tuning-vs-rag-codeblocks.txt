```
파인튜닝: 도메인 톤/스타일 + 전문 용어 이해
    +
RAG: 최신 데이터 + 구체적 사실 정보 + 출처 추적
    =
도메인 전문가 수준의 최신 정보 기반 응답
```
```
최신 정보가 중요한가?
├── 예 → RAG (또는 하이브리드)
└── 아니오
    ↓
출처 추적이 필요한가?
├── 예 → RAG (또는 하이브리드)
└── 아니오
    ↓
특수한 출력 형식/스타일이 필요한가?
├── 예 → 파인튜닝 (또는 하이브리드)
└── 아니오
    ↓
프롬프트 엔지니어링만으로 충분한가?
├── 예 → 프롬프트 엔지니어링
└── 아니오 → 하이브리드
```
