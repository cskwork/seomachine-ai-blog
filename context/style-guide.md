# Style Guide - AI Insights Lab

## 문체
- 경어체 사용 (~합니다, ~입니다)
- 간결한 문장 (한 문장 40자 이내 권장)
- 능동태 선호

## 포맷팅
- 코드 블록: 언어 명시 (```python, ```bash)
- 리스트: 3개 이상이면 불릿 포인트 사용
- 표: 비교 항목이 3개 이상이면 테이블 사용
- 인용: 외부 자료 인용 시 출처 명시

## 기술 용어 규칙
- 첫 등장: "RAG(Retrieval-Augmented Generation, 검색 증강 생성)"
- 이후: "RAG"만 사용
- 고유 명사: 공식 표기 준수 (Claude, GPT-4, LangChain)
